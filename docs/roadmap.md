# Product Roadmap

## Phase 0 — UI Foundation
- Monorepo setup
- App shell + dashboard routes
- Shared packages for core/db/policy abstractions

## Phase 1 — Auth + Tenancy
- Authentication flows
- Tenant-aware session model
- Role assignment and policy enforcement

## Phase 2 — Telemetry Ingestion
- HTTP ingestion pipelines
- Normalized telemetry storage interfaces
- Basic stream validation and retention policy

## Phase 3 — Alarms + Rules
- Alarm rule engine
- Acknowledgement lifecycle
- Notification channels and escalation policies

## Phase 4 — Edge + Integrations
- Edge runtime services
- MQTT/Modbus/OPC UA connectors
- Secure sync between site and cloud

## Phase 5 — Analytics + AI
- Aggregated KPI dashboards
- Forecasting and anomaly insights
- Operator copilots and recommended actions
